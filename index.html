Windows Defender / Antivirus Warning — Expected & Safe
Windows Defender may flag MeteoristAddon1_21_11.jar as a potential threat. This is a false positive. The JAR contains an HWID (Hardware ID) lock — a security feature that binds the addon to your device to prevent redistribution and piracy. This is the same technology used by paid software to protect licenses. It does not steal your data, it only reads your hardware ID to verify you downloaded from the official source. If Defender blocks it: open Windows Security → Virus & threat protection → Protection history → find the file → click Allow.

Do NOT skip OpSec Mod — it prevents your account from being flagged on DonutSMP.

Help

FAQ

Everything you need to know before installing.

General

What is Meteorist?
Meteorist is a free open-source Meteor Client addon for Minecraft 1.21 with 25+ modules built for DonutSMP, including exclusive Deepslate Bypass and Spawner Detection.
Is it free?
Yes. No paid tiers, no premium modules, no subscriptions. Always free.
Does it contain a virus?
No. Meteorist is open-source. The reason Windows Defender may flag it is because the JAR contains an HWID lock — a device-binding security feature that prevents piracy. It reads your hardware ID to verify you downloaded from the official source. It does not steal data. To allow it: Windows Security → Virus & threat protection → Protection history → find the file → Allow.
What Minecraft version?
Minecraft 1.21 with Fabric mod loader. Works with any launcher that supports Fabric — Prism, MultiMC, ATLauncher, TLauncher, vanilla launcher and more.
Does it work with my launcher?
Yes — Meteorist works with ANY launcher that supports Fabric mods. This includes Prism Launcher, MultiMC, ATLauncher, GDLauncher, TLauncher, CurseForge App, and the official vanilla Minecraft launcher. If your launcher can run Fabric, it can run Meteorist.

Installation

Do I need OpSec Mod?
Yes — absolutely. It masks the patterns DonutSMP's anti-cheat watches for. Not optional.
Meteorist isn't showing in Click GUI — help?
Confirm all four JARs are in your mods folder: Meteor Client, OpSec Mod, Fabric API, MeteoristAddon.jar. Make sure you launched the Fabric profile.
Windows Defender blocked the file — what do I do?
This is expected. The JAR has an HWID lock for anti-piracy. To allow it: open Windows Security → Virus & threat protection → Protection history → find MeteoristAddon.jar → click Allow on device. The file is safe.
Where is my mods folder?
Windows: Win+R → %appdata%\.minecraft\mods. Mac: ~/Library/Application Support/minecraft/mods. Linux: ~/.minecraft/mods. For Prism/MultiMC, right-click your instance → Folder → mods.

Features

How does Deepslate Bypass work?
Meteorist's nuker modules are tuned for deepslate's higher hardness, calculating break patterns far exceeding vanilla mining speed underground.
How does Spawner Detection find bases?
Nerd Vision ESP highlights spawner blocks through terrain. On DonutSMP, skeleton spawners are commonly near player bases.
